## 🚀 Featured Project: *StAnify*
**StAnify** is a modular multi-agent AI tutoring system built as my Master's Thesis at the University of Birmingham.
- Dedicated agents for analogy generation, narrative building, evaluation, and image critique
- **RAG pipeline** over structured educational content using **Qdrant** vector search + vocabulary scaffolding
- Integration of **Stable Diffusion XL** and **OpenAI GPT-4o/5** via **CrewAI** multi-agent workflows
- Reduced teacher preparation time from **50–80 mins → ~3 mins** (17–25× faster) at ~$0.15 per concept
🏆 Preferred in **65% of blinded A/B tests** vs GPT-4 — outperforming on clarity (+1.59) and engagement (+1.51)  
🧩 Tech stack: Python · CrewAI · OpenAI API · Stable Diffusion XL · Qdrant · Streamlit · FastAPI
